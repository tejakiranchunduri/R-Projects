1. You can download and unzip data from 'http://utdallas.edu/~sxa172830/train.zip' 
	 or optionally Run 'getData.r' file. Downloads data and extracts it.

2. Run 'main.r'. This contains main code file.

3. Run 'cloud.r'. 
  This will submit the code to google cloud, along with hyper paramter tuning.

4. tuning.yml: 
   Config file for hyper parameter tuning. Contains grid of model parameters.
